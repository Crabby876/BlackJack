# Blackjack 21 – Multiplayer Web App

## About the Project
This project is an interactive, real-time Blackjack application designed specifically for playing together in the browser. It allows players to create private lobbies, invite friends, and compete against each other and the dealer in classic casino fashion.

This project was developed as a collaborative group effort by three people.
